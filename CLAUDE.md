# LLEVANOS.COM - Project Memory

## Overview
Plataforma de venta de boletos para transporte en México. Sitio en construcción con diseño neobrutalista.

## Tech Stack
- **Frontend**: HTML5 + CSS3 + Vanilla JavaScript
- **Styling**: Neobrutalism (yellow #FFE500 + black #0a0a0a)
- **Typography**: Archivo Black (headers) + Space Grotesk (body)
- **Animations**: CSS + IntersectionObserver + requestAnimationFrame
- **Hosting**: Cloudflare Pages (planned)

## Design System

### Colors
- Primary: `#FFE500` (Yellow)
- Background: `#0a0a0a` (Black)
- Secondary: `#2a2a2a` (Dark Gray)
- Text: `#ffffff` (White)

### Components
- **Buttons**: 4px border, hard shadows (8px 8px 0), hover lift effect
- **Cards**: Dark bg, yellow slide-up on hover, 4px border
- **Marquee**: Yellow bg, rotated -1deg, infinite scroll
- **Stats**: Yellow cards with black text, hover rotation

## Features Implemented
1. Custom cursor with follower effect
2. Glitch text animation (hero title)
3. Floating elements with mouse parallax
4. Infinite marquee with scroll-speed modulation
5. Feature cards with hover animations
6. Animated stat counters
7. Newsletter section
8. Scroll reveal animations
9. Responsive design
